## ADDED Requirements

### Requirement: Create Goal UI Styling
The Create Goal screen SHALL be styled according to the new Figma design language.

#### Scenario: Input fields visual style
- **WHEN** the Create Goal screen is displayed
- **THEN** all input fields MUST be fully rounded (pill shape) with a Primary Container background and no visible outline.

#### Scenario: Screen Header
- **WHEN** the Create Goal screen is displayed
- **THEN** a hero graphic MUST be displayed at the top containing a circular primary icon over a translucent circular background.

#### Scenario: Primary Button
- **WHEN** the Create Goal screen is displayed
- **THEN** the primary action button MUST be fully rounded, use the Primary Variant color, and have a colored drop shadow.
